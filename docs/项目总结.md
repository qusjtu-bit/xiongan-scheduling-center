## 技术实现

### 前端技术栈
- HTML5 + CSS3 + JavaScript (ES6+)
- ECharts 5.4.3（CDN引入）
- Font Awesome 6.4.0（CDN引入）
- 纯前端实现，无需框架编译

### 后端技术栈
- Python 3.x + Flask 2.x
- Flask-CORS（跨域支持）
- Mock数据完整丰富

### 设计风格
- **前沿科技风**：深色主题、科技蓝主色调、渐变配色
- **清爽简洁**：信息密度适中、合理留白
- **卡片式布局**：微妙阴影和边框发光效果
- **响应式设计**：适配1920x1080大屏
